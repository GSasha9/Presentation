replaceState and pushState

Method replaceState replaces the current URL in the address bar


				history.replaceState(null, null, 'hello');

*The replaceState method above switches out the URL in the address bar with ‘/hello’ despite no assets being requested and the window remaining on the same page

But there is another method that adds URL to the browser’s history

pushState method changes our history to include whatever URL we just passed into it 


				history.pushState(null, null, 'hello');

Parameters


						history.pushState(null, null, 'hello');

Parameters


					  history.pushState([data], null, 'hello');
  • Data - it is any JavaScript object (or value) that you want to associate with the current state of the page.

Parameters


					  history.pushState([data], [title], 'hello');
  • Data - it is any JavaScript object (or value) that you want to associate with the current state of the page.
  • Title. This parameter exists for historical reasons, and cannot be omitted; passing an empty string is safe against future changes to the method.

Parameters


					  history.pushState([data], [title], [url]);
  • Data - it is any JavaScript object (or value) that you want to associate with the current state of the page.
  • Title. This parameter exists for historical reasons, and cannot be omitted; passing an empty string is safe against future changes to the method.
  • Final parameter is the URL we want to appear in the address bar
